exgcd的用法

2020-05-28

exgcd⁡\operatorname{exgcd}exgcd就是求不等式a×x+b×y=gcd⁡(a,b)a\times x+b\times y=\gcd(a,b)a×x+b×y=gcd(a,b)的其中一组整数解xxx,yyy的方法。

代码长这样：

void exgcd(int a,int b,int &x,int &y)
{
	if(b==0)
	{
		x=1,y=0;
		return;
	}
	exgcd(b,a%b,x,y);
	int t;
  	t=x;
	x=y;
  	y=t-a/b*y;
}

当b=0b=0b=0时aaa就是这两个数的最大公约数，所以x=1,y=0x=1,y=0x=1,y=0时就满足等式。

即b×x+(a mod b)×y=gcd⁡(a,b)b\times x+(a \bmod b)\times y=\gcd(a,b)b×x+(amodb)×y=gcd(a,b)
